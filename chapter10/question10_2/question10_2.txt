question 10_2

次のような「社員クラス」と「部署クラス」があるとします。

// 社員クラス
class Employee {
	String name;
	int age;
}
// 部署クラス
class Department {
	String name;
	Employee leader;
}

「総務部」のリーダー「田中太郎（41歳）」のインスタンスをJVM内に生成したうえで、
直列化機構を使ってファイルC:\javatest\company.datに書き込むプログラムを作成してください。
なお、上記2つのクラスを必要な範囲で修正してください。



■解説

・直列化を可能にするにはjava.io.Serializableインタフェースを使います
・オブジェクトファイルを書き込むにはObjectOutputStreamクラスを使います
・writeObjectメソッドを使って、存在するオブジェクトをそのまま書き込みます
