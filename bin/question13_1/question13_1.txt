question13_1

以下のJavaソースファイルを作成し、変数名からクラスやメンバの意味を推測して、
適切なJavadocコメントとタグを書き加え、javadocツールでプログラム仕様書を生成してください。

public class Bank {
	String name;
	String address;
	public void addAccount(String owner, int initialBalance) { }
	public static void main(String[] args) {
		System.out.println("試験用のメインメソッドです");
	}
}

Eclipseでの手順：
	1. パッケージ（プロジェクト）を選択して右クリックします。
	2. 「エクスポート」⇒「Javadoc」を選択します。
	3. エクスポート先を選択します。
	4. 「完了」を選択します。



■解説

・ソースにコメントとタグを記述することで、プログラムの内容をわかりやすくします

・Javadocコメントの書き方

	Javadocコメントを書く場合は「/**」と「*/」の間にコメントを記述します

	/** Javadocコメント */

	/**
	 * Javadocコメント
	 * 複数行はこのように記述します
	 */

	Javadocコメントの中で「@」を使うとタグが付けられます
	この問題では以下のタグを適切な位置に記述してください

	@param メソッドの引数の説明
